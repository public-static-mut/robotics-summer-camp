What the summer is aiming for

  • FTC boot camp for new folks and rookies
  • Basic CAD and useful 3D-printed parts
  • Programming practice with GitHub and Pedropathing

The summer plan

Designed for rookies, returning members, and anyone who wants to get ahead.

The idea is to keep things practical: build a bot, understand the electronics, make parts that matter, and work through code with a real team workflow.

Week 01

FTC Boot Camp

A crash course for new folks: free-build, wire, code a basic bot, and get comfortable with drivetrain, intake, teleop, and auto.

Week 02

Basic CAD Camp

Rapid iteration in OnShape: servo claws, intakes, dumpers, and other useful 3D-printed parts that connect to the build.

Week 03

Programming Camp

For advanced rookies and returning members: Android Studio, Pedropathing, version control, and code study from excellent FTC examples.

Week 04

Advanced CAD Camp

Master sketches, full robot design, and version control. A deeper look at how teams plan before they build and keep ideas organized.

Flexible

Early Season Sessions

Optional once- or twice-a-week meetups to start next season early with new code, a new chassis, or a useful intake for the next game.

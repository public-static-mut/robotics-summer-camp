A few things we could do

Built for rookies, returning members, and anyone who wants to get ahead a little.

The goal is pretty simple: keep it useful, not overcomplicated, and make sure everyone leaves with something they can actually use next season.

Week of July 6

FTC Boot Camp

This is the crash course idea for new folks, including kids from Western Edge. Free-build, basic electronics, a simple drivetrain, and enough code to move the robot around in teleop and auto.

Basic CAD

Useful parts, not giant assemblies

We would keep this pretty practical. Servo claws, intakes, dumpers, and other printed parts that actually help tie the robot together. The idea is CAD, print, improve, repeat.

Programming

Programming Camp

For the advanced rookies and returning members. Android Studio, Pedropathing, GitHub, and looking through good FTC code so everyone can start understanding how the better teams do it.

Advanced CAD

Planning the whole robot before building it

This would be more like the Samotech level of planning: master sketches, whole-robot design, and version control so we can keep moving without getting lost.

Optional

Early Season Sessions

Another option is to just start the season early and meet once or twice a week to work on code, a new chassis, or whatever else looks most useful for next year.
